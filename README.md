For my project, I chose to do the number guessing game. The game works by first prompting the player to input a difficulty for the game. If the player picks “easy”, a random number from 1 to 100 will generate. Selecting “medium” will generate a number 1 to 1000 and selecting hard will generate a number 1 to 10000. If the player does not enter a valid difficulty, they will be prompted to enter a valid difficulty. Once a difficulty is selected, a number is generated and the user enters in their guesses. If their guess is too large, a message saying “guess is too large” will appear. If their guess is too small a message saying “guess is too small” will appear. Once the player has guessed the number a message saying, “You guessed it in x tries!” will appear. The player will then be asked if they want to play again. If they type yes, the whole process will repeat. If they type no, the game will end and the number of guesses it took to guess the number for each game will be displayed.

To run the progam, press the run button and look at the terminal. In the terminal, the player will be prompted to enter a diffuculty. Then, they will be prompted to enter a guess. The player will continue guessing until the numnber is guessed. Once the number is guessed, the player will be propted if they want to play the game again. If they type yes, a new round will start. If they type no, the game will end and the game stats will be displayed.
